Setting up a development environment with typescript can be challenging because the tools are rapidly evolving and it’s hard to syncrhonize between different tutorials. As such, I am stating the versions that I’m using now, and you should search for better new methods whenever these tools become out of date.

Tools that you’ll need: * Visual Studio Code >= 1.13.1 * Node.Js >= 8.1.2 * NPM >= 5.0.2 * Yarn >= 0.24.5

This tutorial assumes the following dependency versions: * Typescript 2.2 * Tslint 4.5.1

If you come from another language, search for your favorite key mapping extensions for Visual Studio Code.

Another useful typescript plugin that you could use is tslint.
